#### Full Form 
```mysql
CASE
	WHEN <condition1> THEN <result1>
	WHEN <condition2> THEN <result2>
	...
	ELSE result
END
```

#### Quick Form 
```mysql 
CASE Column
	WHEN <VALUE> THEN <result1>
	WHEN <VALUE> THEN <result2>
	...
	ELSE result
END
```
