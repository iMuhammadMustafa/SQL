# WHERE

## Comparison Operators

- Equal: `=`
- Not Equal: `!=` or `<>`
- Greater Than: `>`
- Less Than: `<`
- Greater Than or Equal To: `>=`
- Less Than or Equal To: `<=`

```sql
SELECT * FROM Persons WHERE Age = 30;
SELECT * FROM Persons WHERE Age != 30;
SELECT * FROM Persons WHERE Age > 30;
SELECT * FROM Persons WHERE Age < 30;
SELECT * FROM Persons WHERE Age >= 30;
SELECT * FROM Persons WHERE Age <= 30;
```

## Logical Operators

- AND
- OR
- NOT

```sql
SELECT * FROM Persons WHERE Age > 30 AND City = 'New York';
SELECT * FROM Persons WHERE Age > 30 OR City = 'New York';
SELECT * FROM Persons WHERE NOT Age > 30;
```

## Range Operators

- BETWEEN ... AND ...
- NOT BETWEEN ... AND ...

```sql
SELECT * FROM Persons WHERE Age BETWEEN 20 AND 30;
SELECT * FROM Persons WHERE DateOfBirth NOT BETWEEN '1990-01-01' AND '2000-12-31';
```

## Membership Operators

- IN (...)
- NOT IN (...)

```sql
SELECT * FROM Persons WHERE City IN ('New York', 'Los Angeles', 'Chicago');
SELECT * FROM Persons WHERE City NOT IN ('New York', 'Los Angeles', 'Chicago');
```

## Pattern Matching Operators

- LIKE

```sql
SELECT * FROM Persons WHERE LastName LIKE 'S%'; -- Last names starting with 'S
SELECT * FROM Persons WHERE LastName LIKE '%son'; -- Last names ending with 'son'
SELECT * FROM Persons WHERE LastName LIKE '%a%'; -- Last names containing 'a'
SELECT * FROM Persons WHERE LastName LIKE 'S_m%'; -- Last names starting with 'S', followed by any single character, followed by 'm'
```

- \_ (underscore) matches any single character

```sql
SELECT * FROM Persons WHERE LastName LIKE 'S_m%'; -- Last names starting with 'S', followed by any single character, followed by 'm'
-- This will match last names like 'Smith', 'Samm', 'S_mith', etc.

```

- % (percent) matches any sequence of characters (including zero characters)

```sql
SELECT * FROM Persons WHERE LastName LIKE 'S%'; -- Last names starting with 'S'
-- This will match last names like 'Smith', 'Simmons', 'S', etc.
SELECT * FROM Persons WHERE LastName LIKE '%son'; -- Last names ending with 'son'
-- This will match last names like 'Johnson', 'Wilson', 'son', etc.
```
